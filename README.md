# UE4-Discord-Rich-Presence
Unreal Engine 4 Discord Rich Presence Plugin

> Blueprint Nodes
- Create Discord Object
- Destroy Discord Object
- Get Discord Object
- Set Details
- Set State
- Set Large Image
- Set Large Image Text
- Set Small Image
- Set Small Image Text
- Set Party Id
- Set Party Max
- Set Party Size
- Set Join Secret
- Start Discord Timer
- Stop Discord Timer
